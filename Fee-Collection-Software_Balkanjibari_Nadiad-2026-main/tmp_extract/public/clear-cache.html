🔐 Clear Authentication Cache

Issue: Your browser has cached authentication tokens, causing the admin panel to show directly instead of the login page.

Quick Fix (Recommended)

Manual Steps

Option 1: Use Incognito/Private Mode
  1. Open a new incognito/private window
  2. Navigate to http://localhost:3000
  3. You should see the login page
Option 2: Clear Browser Storage
  1. Press F12 to open Developer Tools
  2. Go to Application tab (Chrome) or Storage tab (Firefox)
  3. Click on Local Storage → http://localhost:3000
  4. Delete access_token and refresh_token
  5. Refresh the page
Option 3: Hard Refresh
  1. Press Ctrl + Shift + Delete (Windows) or Cmd + Shift + Delete (Mac)
  2. Select "Cached images and files" and "Cookies and other site data"
  3. Click "Clear data"
  4. Navigate to http://localhost:3000
